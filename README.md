Hi, I'm Shane. I build practical tools with Python, computer vision, and automation.

## Featured Work

### YoloPU

A computer vision experiment project for reading video or real-time screen input, detecting human keypoints with YOLO Pose, and displaying posture as a stick-figure skeleton.

It includes:

- PyQt UI with capture preview, model selection, and experiment controls
- YOLO detection viewer for selected application windows
- Hand landmark detection with MediaPipe Hands
- HSV color tracking for comparison experiments
- LocateAnything screenshot and local image positioning
- Safe YOLO demo flow for dataset generation, training, detection, center-point conversion, and click-result reporting
